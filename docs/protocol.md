# WebSocket 通信协议

## 基本信息

| 项目 | 值 |
|------|------|
| 传输协议 | WebSocket |
| 数据格式 | JSON 文本消息 |
| 编码 | UTF-8 |
| 角色 | 小车 = Server，PC = Client |
| 默认端口 | 9001 |

每条消息为单行 JSON，顶层必有 `type` 字段。

---

## 上行：小车 → PC

### pose — 车辆位姿

实时发送车辆位置、朝向和速度。

```json
{
    "type": "pose",
    "ts": 1717800000.123,
    "x": 1.5,
    "y": 0.0,
    "z": 3.2,
    "yaw": 0.785,
    "vx": 0.5,
    "vz": 0.0
}
```

| 字段 | 类型 | 单位 | 说明 |
|------|------|------|------|
| `ts` | f64 | 秒 | Unix 时间戳 |
| `x`, `y`, `z` | f32 | 米 | 世界坐标 |
| `yaw` | f32 | 弧度 | 偏航角 |
| `vx`, `vz` | f32 | 米/秒 | 速度分量 |

---

### voxel_full — 全量体素

连接建立或重连后发送完整地图。

```json
{
    "type": "voxel_full",
    "ts": 1717800000.200,
    "voxels": [
        {"gx": 0, "gy": 0, "gz": 0, "state": 1, "conf": 0.95},
        {"gx": 1, "gy": 0, "gz": 0, "state": 2, "conf": 0.80}
    ]
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `ts` | f64 | Unix 时间戳 |
| `voxels` | array | 全量体素列表 |
| `gx`, `gy`, `gz` | i32 | 网格坐标 |
| `state` | u8 | 0=未知 1=空闲 2=占用 |
| `conf` | f32 | 置信度 0.0~1.0 |

---

### voxel_delta — 增量体素

仅发送变化的格子。

```json
{
    "type": "voxel_delta",
    "ts": 1717800000.300,
    "voxels": [
        {"gx": 2, "gy": 0, "gz": 1, "state": 2, "conf": 0.90}
    ]
}
```

字段同 `voxel_full`。

---

### path — 规划路径

```json
{
    "type": "path",
    "ts": 1717800000.400,
    "points": [
        {"x": 0.0, "z": 0.0},
        {"x": 1.0, "z": 1.5},
        {"x": 3.0, "z": 4.2}
    ]
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `ts` | f64 | Unix 时间戳 |
| `points` | array | 路径点序列，顺序 |
| `x`, `z` | f32 | 世界坐标 |

---

## 下行：PC → 小车

### ctrl — 键盘控制

按下或释放按键时发送。

```json
{
    "type": "ctrl",
    "key": "w",
    "action": "press"
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `key` | string | `w` `s` `a` `d` `space` |
| `action` | string | `press` 按下 / `release` 释放 |

| 按键 | 动作 | 说明 |
|------|------|------|
| `w` | 前进 | 沿车头方向移动 |
| `s` | 后退 | 沿车头反方向移动 |
| `a` | 左旋 | 车身原地左转（逆时针） |
| `d` | 右旋 | 车身原地右转（顺时针） |
| `space` | 🛑 紧急停止 | 速度归零 |

> 控制方式为**坦克式操纵**：A/D 原地旋转车身朝向，W/S 沿朝向直线移动。
